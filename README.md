# Sistema de Inventario YAVIRAC

Sistema de gestión de inventario para el Superior Tecnológico de Turismo y Patrimonio YAVIRAC.

## Características

- **Login**: Sistema de autenticación con diseño moderno
- **Inventario**: Gestión completa de bienes con tabla interactiva
- **Modales**: Agregar nuevos bienes y ver detalles
- **Diseño Responsive**: Adaptable a diferentes tamaños de pantalla

## Instalación

1. Instalar dependencias:
```bash
npm install
```

2. Ejecutar el servidor de desarrollo:
```bash
npm start
```

3. Abrir en el navegador:
```
http://localhost:4200
```

## Estructura del Proyecto

- `src/app/components/login/` - Componente de login
- `src/app/components/inventario/` - Componente de inventario
- `src/app/components/sidebar/` - Componente de barra lateral
- `src/assets/` - Recursos estáticos (logo, imágenes)

## Tecnologías

- Angular 17
- TypeScript
- SCSS
- RxJS
